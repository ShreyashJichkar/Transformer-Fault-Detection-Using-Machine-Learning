# Transformer Fault Detection Using Machine Learning

## Overview

This project demonstrates the application of Machine Learning techniques for transformer fault detection using Dissolved Gas Analysis (DGA) parameters. The objective is to classify transformer operating conditions and identify potential faults based on gas concentration values.

The project is implemented in Python using Jupyter Notebook and follows a complete Machine Learning workflow including data preprocessing, exploratory data analysis, model training, evaluation, and fault prediction.

## Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Fault Categories

* Normal Condition
* Partial Discharge
* Thermal Fault
* Arcing Fault

## Features

* Data Generation and Preprocessing
* Exploratory Data Analysis (EDA)
* Random Forest Classification
* Model Evaluation
* Confusion Matrix Visualization
* Feature Importance Analysis
* Real-Time Fault Prediction

## Machine Learning Workflow

1. Generate and prepare DGA data
2. Perform exploratory data analysis
3. Visualize fault distributions
4. Split data into training and testing sets
5. Train Random Forest model
6. Evaluate model performance
7. Predict transformer fault conditions

## Results

The model successfully classifies transformer conditions based on DGA parameters and demonstrates the complete workflow of a Machine Learning classification project.

> Note: This project uses a simulated DGA dataset created for educational and demonstration purposes.
