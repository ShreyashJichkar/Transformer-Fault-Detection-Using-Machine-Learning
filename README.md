# Transformer-Fault-Detection-Using-Machine-Learning
This project focuses on the development of a Machine Learning-based system for detecting and classifying faults in power transformers using Dissolved Gas Analysis (DGA) data. Power transformers are among the most important and expensive assets in electrical power systems,
This project focuses on detecting and classifying transformer faults using Machine Learning techniques. Transformer health is monitored through Dissolved Gas Analysis (DGA) parameters such as Hydrogen (H₂), Methane (CH₄), Acetylene (C₂H₂), and Ethylene (C₂H₄). By analyzing these gas concentrations, the model predicts the type of fault occurring inside the transformer.

The project demonstrates the complete Machine Learning workflow, including data generation, preprocessing, exploratory data analysis, model training, evaluation, and fault prediction. A Random Forest Classifier is used to classify transformer conditions into different fault categories.

🎯 Objectives
Detect transformer faults using DGA parameters.
Classify transformer conditions into fault categories.
Apply Machine Learning techniques for predictive maintenance.
Visualize and analyze transformer fault data.
Improve understanding of AI applications in power systems.
